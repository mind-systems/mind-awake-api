# Breath Sessions — Complexity

Каждая сессия дыхания получает числовое значение `complexity` при создании и обновлении. Оно рассчитывается автоматически на основе структуры упражнений.

## Формула

```
complexity = max(0, contribution − penalty)
```

### Contribution (вклад)

Суммарное время активной работы по всем упражнениям:

```
contribution = Σ (cycleDuration × repeatCount)
```

где `cycleDuration` — сумма `duration` всех шагов (`inhale`, `exhale`, `hold`) одного упражнения.

### Penalty (штраф)

Отдых уменьшает сложность. Два вида штрафов:

| Тип отдыха | Условие | Формула |
|------------|---------|---------|
| **Между сетами** | `exercise.restDuration > 0` и `steps ≠ []` | `restDuration × repeatCount × 5` |
| **Между упражнениями** (rest separator) | `exercise.steps = []` и `index > 0` | `restDuration × 3` |

> Rest separator на позиции `index = 0` (вступительная пауза) — не штрафуется.

Результат не может быть ниже `0`.

## Примеры

### Простое упражнение без отдыха

```
inhale 4s + exhale 4s, без отдыха, ×3
```

| | Значение |
|---|---|
| contribution | `(4000 + 4000) × 3 = 24000` |
| penalty | `0` |
| **complexity** | **24000** |

### Упражнение с отдыхом между сетами

```
inhale 4s + hold 2s + exhale 4s, отдых 1s, ×4
```

| | Значение |
|---|---|
| contribution | `(4000 + 2000 + 4000) × 4 = 40000` |
| penalty | `1000 × 4 × 5 = 20000` |
| **complexity** | **20000** |

### Составная сессия

```
[0] Rest separator: 5s          — index 0, без штрафа
[1] inhale 4s + exhale 4s, отдых 1s, ×3
[2] Rest separator: 2s          — index > 0, штраф
[3] inhale 3s, без отдыха, ×2
```

| Элемент | contribution | penalty |
|---------|-------------|---------|
| [0] Вступительная пауза 5s | — | — |
| [1] Упражнение `(4+4s) × 3` | 24000 | `1000 × 3 × 5 = 15000` |
| [2] Пауза между упражнениями 2s | — | `2000 × 3 = 6000` |
| [3] Упражнение `3s × 2` | 6000 | — |
| **Итого** | 30000 | 21000 |

```
complexity = max(0, 30000 − 21000) = 9000
```

## Когда рассчитывается

- При **создании** сессии (`POST`)
- При **обновлении** сессии (`PATCH`, `PUT`) — пересчитывается, если изменились `exercises`

Значение хранится в колонке `complexity` (тип `float`) таблицы `breath_sessions`.
